# Machine Learning Algorithms

Ce dépôt contient des implémentations pédagogiques des principaux algorithmes de machine learning en Python.  
Chaque algorithme est accompagné de démonstrations (Jupyter notebooks) et parfois de petits datasets pour tester rapidement.

---

## 📂 Structure du projet
- `algorithms/` : implémentations des algorithmes
- `notebooks/` : notebooks interactifs pour visualiser les résultats
- `datasets/` : exemples de jeux de données
- `utils/` : fonctions utilitaires (prétraitement, métriques)
